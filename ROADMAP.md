# Priorities Game - Feature Roadmap

This document tracks planned features and enhancements for future development phases.

---

## Phase 3: Polish & Enhancement

### 3.1 - Game Polish
- [ ] Better animations (card reveals, score updates)
- [ ] Sound effects (optional)
- [ ] Confetti/celebration on win
- [ ] Better mobile UX tweaks
- [ ] Loading states polish

### 3.2 - User Accounts (Optional Registration)
- [ ] Anonymous play remains default
- [ ] Option to create account (email/Google)
- [ ] Save game history
- [ ] Player stats (games played, win rate)

### 3.3 - Custom Card Decks
- [ ] Users can create custom decks (requires account)
- [ ] Public/private deck options
- [ ] Browse community decks
- [ ] Minimum 50 cards per deck validation
- [ ] Host selects deck in lobby

### 3.4 - Social Features
- [ ] In-game chat during guessing phase
- [ ] Reactions/emojis
- [ ] "Play again" with same group
- [ ] Invite links with room codes

### 3.5 - Host Powers
- [ ] Kick players
- [ ] Change settings (win threshold, timer)
- [ ] Pause/resume game

### 3.6 - Game Variations
- [ ] Different scoring modes
- [ ] Timed rounds (optional)
- [ ] Tournament mode (multiple games)

---

## Phase 4: Advanced Features

### 4.1 - Seasonal/Topical Cards
- [ ] Cards that refresh based on current events
- [ ] API integration for trending topics
- [ ] Holiday-themed decks

### 4.2 - Analytics
- [ ] Track popular cards
- [ ] See which cards cause most debate
- [ ] Heatmaps of rankings

### 4.3 - Monetization (if desired)
- [ ] Premium card packs
- [ ] Custom themes
- [ ] Ad-free experience

---

## Notes

- This roadmap is subject to change based on user feedback and priorities
- Features will be implemented iteratively, with user testing between phases
- Community feedback will help determine which features to prioritize
